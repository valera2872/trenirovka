# Сила рук BJJ

Офлайн‑трекер дополнительной силовой подготовки рук для 14‑летнего спортсмена, занимающегося бразильским джиу‑джитсу.

## Возможности версии 0.1

- две чередующиеся тренировки A/B;
- отметка каждого выполненного подхода;
- индивидуальный вес, число подходов и повторений для каждого упражнения;
- оценка боли, техники и субъективной тяжести после занятия;
- двойная прогрессия: сначала повторения, затем минимальный шаг веса;
- облегчённый режим после тяжёлого BJJ или недостаточного сна;
- замеры плеча, предплечья и массы тела;
- история тренировок и график динамики;
- два еженедельных локальных напоминания;
- все данные хранятся только на телефоне.

## Базовая схема

Две дополнительные тренировки в неделю, между ними не менее 48 часов. Подходы выполняются не до отказа: в конце рабочего подхода должен оставаться запас примерно 2–3 технически чистых повторения.

## Ограничение

Приложение не заменяет очную оценку спортивного врача, физиотерапевта или квалифицированного тренера. При боли в суставе, онемении, резкой слабости, отёке либо необычной боли тренировку нужно прекратить.
