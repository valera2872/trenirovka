# Сила рук BJJ

Мобильный офлайн‑трекер дополнительной силовой подготовки рук для 14‑летнего спортсмена, занимающегося бразильским джиу‑джитсу.

## Что делает приложение

- чередует две короткие тренировки A/B;
- показывает подходы, повторения, вес и технику;
- отмечает выполненные подходы;
- после тренировки спрашивает о боли, качестве техники и субъективной тяжести;
- повышает сначала повторения, затем минимально доступный вес;
- не повышает нагрузку при боли, плохой технике или недостаточном восстановлении;
- хранит замеры плеча, предплечья и массы тела;
- показывает историю и динамику;
- напоминает о двух тренировках в неделю.

## Важное ограничение

Приложение не заменяет очную оценку врача, физиотерапевта или квалифицированного тренера. При боли в суставе, онемении, резкой слабости, отёке или боли, сохраняющейся дольше обычной мышечной крепатуры, тренировку нужно прекратить.
